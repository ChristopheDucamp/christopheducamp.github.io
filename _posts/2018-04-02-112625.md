Après [OwnYourGram](http://ownyourgram.com/) pour poster automatiquement les photos instagram, j'aimerais goûter à d'autres [apps tiers sur Micro.blog](http://help.micro.blog/2017/micropub-clients/) afin de rentrer proprement dans les conversations.

Par ordre de priorité, j'envisage de tester et vous prie d'avance de m'excuser sur la pollution de la timeline : 

* [Quill](https://quill.p3k.io/) — client navigateur web qui peut poster sur Micro.blog et téléverser des photos.  
* [Omnibear](https://chrome.google.com/webstore/detail/omnibear/cjieakdeocmiimmphkfhdfbihhncoocn) — une extension Chrome pour poster.
* [TodayPoster](https://github.com/bryanluby/TodayPoster) — macOS Today Widget pour poster.

## C'est quoi Micropublish ? 

<dfn>[Micropublish](https://github.com/barryf/micropublish)</dfn> est une simple web app open source, pour poster mettre à jour et effacer du contenu via [[Micropub]](https://indieweb.org) sur un site web.

Micropublish a été écrit par [Barry Frost](https://indieweb.org/User:Barryfrost.com) qui héberge une version live sur [micropublish.net](https://micropublish.net). 

Tout le [code source](https://github.com/barryf/micropublish) est diponible sur GitHub.
